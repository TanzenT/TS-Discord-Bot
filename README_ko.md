## 설치

> yarn 사용시

1. 이 프로젝트를 포크한 뒤, 컴퓨터에 복제하세요.
2. 터미널을 열고 `yarn install`을 실행하세요.
3. `config.example.json`파일의 이름을 `config.json`로 변경하세요.
4. `config.json` 파일을 구성하세요.
5. `yarn start` 명령어를 실행하세요.

> npm 사용시

1. 이 프로젝트를 포크한 뒤, 컴퓨터에 복제하세요.
2. 터미널을 열고 `npm i`을 실행하세요.
3. `config.example.json`파일의 이름을 `config.json`로 변경하세요.
4. `config.json` 파일을 구성하세요.
5. `npm start` 명령어를 실행하세요.
